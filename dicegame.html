INSTRUCTION'S

Here is a very simple game of sequences or runs. Skills learned: Recognizing numbers and sequences, early strategic thinking, and counting by in five Dice needed: 6 Directions for play: Each player takes a turn rolling the dice and looking for runs starting with (1-2, 1-2-3, and so on). Each dice part of the run scores five points. A player can only have one run in each roll. The first who reaches 100 points is the winner. Examples of play:

Hand 1: Player throws six dice and gets the numbers 1, 4, 2, 1, 3, and 2. The following sequences can be scored: 1, 2 (10 points) or 1, 2, 3, 4 (20 points). The hand is worth 30 points.

Hand 2: Player throws six dice and gets the numbers 1, 4, 6, 6, 3, 3. No sequences can be formed, so no score is recorded.
